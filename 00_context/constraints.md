# Constraints, Decision Rules & Clearance Firewall

## Non‑negotiables
- **Avoid security clearance**: Exclude any company/role that requires, supports, or states “ability/eligibility to obtain” security clearance. This is a hard firewall.
- **Location**: Albuquerque, NM. Remote strongly preferred; hybrid/local acceptable.
- **Role lanes (priority)**:
  1) Internal Tools / Workflow Automation / Integration Engineering (software, not PLC)
  2) Analytics Engineering / Data Engineering (operational data/pipelines)
- **Avoid**: “Automation Engineer” postings that are industrial controls unless explicitly software.

## Compensation Targets (Anchors)
- **Local/Hybrid**: $80k–$110k
- **Fully Remote**: $90k–$130k
- Adjust only if listings show consistent lower/higher market ranges.

## Decision Rules
- **Remote first**: If remote is available, prioritize it over local/hybrid.
- **Business proximity**: Prefer roles close to revenue, internal tooling, or workflow efficiency.
- **Ownership + outcomes**: Favor environments where you can ship end‑to‑end and demonstrate measurable impact.
- **Language agnostic**: Choose stacks based on speed to ship and integration fit.
- **Clearance screening**: If a company is defense‑adjacent or frequently lists clearance requirements, treat it as excluded unless a specific role explicitly states no clearance.

## Clearance Firewall Checklist
Exclude any company/role if you see:
- “Security clearance required”
- “Ability/eligibility to obtain clearance”
- “Must be a U.S. citizen for clearance”
- “Work on classified programs”
- “ITAR/EAR compliance required” (common proxy for defense clearance environments)
- “DoD/IC programs” or “national security mission” (unless explicitly non‑clearance commercial work)

## Role Filtering Heuristics
- **Include**: internal tools, integrations, automation software, ETL/ELT, data platforms, analytics engineering, data quality, workflow orchestration, iPaaS.
- **Exclude**: industrial controls, PLC/SCADA, plant floor automation (unless explicitly software integration/platform).
  - Also exclude: defense contractors, aerospace/space systems, and labs that commonly require clearance unless the posting explicitly says no clearance needed.
