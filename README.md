# Job Acquisition Engine — 90‑Day System

## Weekly Workflow (30–60 min setup + daily execution)
1. **Refresh Targets** (Mon)
   - Review `/01_targets/` and add 3–5 new companies.
   - Remove any clearance‑adjacent listings.
   - Log clearance flags in `/04_outreach/tracking_template.md`.
2. **Search + Apply** (Tue–Thu)
   - Use `/02_search_strings/job_board_queries.md`.
   - Apply to 8–12 targeted roles/week.
3. **Outbound** (Daily)
   - Send 3–5 messages/day from `/04_outreach/scripts_manager_engineer_recruiter.md`.
   - Track in `/04_outreach/tracking_template.md`.
4. **Portfolio Shipping** (Fri)
   - Ship one increment from `/03_portfolio/`.
   - Record a short demo and add to README in each project.
5. **Interview Prep** (Weekend)
   - Follow `/05_execution_plan/interview_prep_loop.md`.

## Rules to Always Enforce
- Never apply to clearance‑bound roles.
- Prefer remote roles; accept ABQ hybrid when aligned.
- Prioritize internal tools/integration or analytics/data engineering roles.
