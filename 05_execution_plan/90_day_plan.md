# 90‑Day Job Acquisition Engine Plan

## Weekly Quotas (baseline)
- **Applications**: 8–12 targeted
- **Outbound**: 15–25 messages
- **Interview reps**: 3–5 (behavioral + system design + SQL)
- **Portfolio shipping**: 1 demo slice/week

## Week‑by‑Week
### Weeks 1–2: Foundation
- Finalize target lists + search strings.
- Build Flagship MVP skeleton (ingest → transform → route).
- Create Proof 1 (connector sync) basic version.
- Start outbound: 15–20 messages/week.

### Weeks 3–4: Signal Boost
- Add admin UI + run logs.
- Publish demo videos + READMEs.
- Apply to 20–25 roles across target list.
- Start interview loop (SQL + system design).

### Weeks 5–6: Reliability + Scale
- Add retries, error alerts, metrics.
- Build Proof 2 (data quality gate).
- Increase outbound to 25/week.
- Track responses and refine messaging.

### Weeks 7–8: Portfolio Polish
- Finish observability pack.
- Tighten docs, add screenshots.
- Mock interviews + review weak spots.

### Weeks 9–10: Conversion Push
- Double down on warm leads.
- Re‑apply or re‑target companies with fresh proof.
- Improve resume with measurable outcomes.

### Weeks 11–12: Close
- Final interview prep.
- Negotiate with anchors.
- Maintain outbound until offer signed.
