# Role Titles & Keyword Packs

## Priority Role Titles
- Internal Tools Engineer
- Workflow Automation Engineer (software)
- Integration Engineer
- Business Systems Engineer
- Platform Engineer (Internal Platforms)
- Analytics Engineer
- Data Engineer
- Data Platform Engineer
- ETL/ELT Engineer
- BI Engineer (if heavy data pipeline focus)
- RevOps/Revenue Systems Engineer (if tooling + integrations)
- Developer Productivity Engineer (internal tooling)

## Keyword Packs (use in searches + resume tailoring)
### Internal Tools / Workflow Automation
- internal tools, admin tools, back‑office automation
- workflow automation, business process automation
- integrations, webhooks, APIs, iPaaS
- developer productivity, tooling, platform

### Integration Engineering
- ETL/ELT, data sync, connectors
- API integrations, middleware
- event‑driven, queues, webhooks
- iPaaS, integration platform, orchestration

### Analytics / Data Engineering
- data pipelines, data warehouse, lakehouse
- SQL, dbt, Airflow, Dagster
- data quality, observability, lineage

## Negative Keywords (filter out)
- security clearance
- “eligible for clearance”
- “ability to obtain clearance”
- PLC, SCADA, DCS, industrial controls
- embedded/firmware (if not software integration)
- “classified”, “national security”, “DoD”, “IC”
